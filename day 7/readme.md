Task 7 Report: Identifying and Removing Suspicious Browser Extensions
1. Introduction
The purpose of this task is to analyze installed browser extensions, identify potentially suspicious or unsafe extensions, evaluate their permissions, and remove those that might pose security or privacy risks.

2. Tools Used
- Google Chrome (Extensions Manager)
- Google search for background verification

3. List of Installed Extensions
Ad Blocker: Stands AdBlocker – Safe
Adblock for YouTube™ – Medium Risk (Unofficial)
Adblocker for YouTube™ – High Risk (Unofficial Clone)
Google Docs Offline – Safe
Google Translate – Safe
PPT Slides Editor – Medium Risk
Presentation Editor PPTWork – High Risk

4. Analysis
Multiple YouTube adblockers found. Unofficial versions often contain tracking scripts or malware. Unknown PPT editors are high-risk due to unverified developers.

5. Recommendations
- Remove both unofficial YouTube adblockers.
- Remove both PPT editing extensions.
- Keep trusted extensions only.

6. Conclusion
Removing untrusted extensions significantly improves browser security and protects against data theft and malicious activity.
